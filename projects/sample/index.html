Splitters / "Nice Tables"

This template has some nice pre-sized column layouts for you to use! You can still build your own tables, OR you can use the following syntax:

<div class="row">

<div class="col-md-N">
...
</div>

</div>

where N = a number 1 to 9 representing N*10 percent of the total width. So for instance, if you want something that takes 30% of the width, you would use "col-md-3". (Is this a convoluted scheme? yeah kinda, I did not make it, so... yeah sorry about that.
